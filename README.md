This is BLUEsat's new QT gui repo.

# Structure

- src/ - all ros files are contained in this folder

 - gui/ - main gui ros package

  - include - C++ header files

  - resource - QML and image files for the GUI

  - src - C++ source files

 - ros_video_components

  - include - C++ header files

  - resource - QML and image files for the GUI

  - src - C++ source files
